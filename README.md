# 🎬 MovieMagic - Movie Recommendation Web App

## 📌 Project Description
MovieMagic is a web-based application built using Flask that allows users to explore and view movies based on different categories such as city and language. The application displays movie details along with posters and provides a simple and interactive interface for users.

---

## 🚀 Features
- 🎥 Display movies with posters and titles
- 🔍 Filter movies based on city and language
- 📊 Backend data handling using Python
- 💾 Database integration using SQLite
- 🌐 Simple and user-friendly web interface

---

## 🛠️ Technologies Used
- Python
- Flask
- HTML
- CSS
- SQLite
