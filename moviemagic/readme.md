# 🎬 Movie Magic – Movie Booking Web App

## 📌 Overview

**Movie Magic** is a user-friendly movie ticket booking web app built with Flask. It allows users to register, log in, view available movies by city/language, book tickets, select seats, and receive booking confirmation.

---

## 🚀 Features

- 🧑‍💻 User Signup & Login
- 🎞️ Movie filtering by city & language
- 🖼️ Poster display from local/static files
- 🎫 Seat selection
- 💳 Simulated payment page
- 📩 Booking confirmation with email

---

## 🛠️ Tech Stack

- **Backend**: Flask (Python)
- **Frontend**: HTML, CSS, Jinja2 Templates
- **Database**: SQLite (`users.db`, `movies.db`)
- **Email**: SMTP (configured in `.env`)
- **Assets**: Stored under `/static/images/`
